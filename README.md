# exemplo-chatterbot
um exemplo simples usando a biblioteca chatterbot!

Requisitos:

Python3 e Pip

Instalação do Chatterbot:
``pip install chatterbot``

Rodando o Projeto:
* Na pasta do projeto abra o terminal e digite: ``python3 chatbot.py``
